## Managed Browser Example

This example shows how to use the Airbnb client with a managed browser. The browser is launched and managed by the client, from inside a docker container


## How to run

1. Run the rod image:
   ```shell
   docker run -p 7317:7317 ghcr.io/go-rod/rod
   ```
2. Run the example:
   ```shell
   go run .
   ```
